MarkDown 常用规则

前言

写blog主要需要用到MarkDown，所以必须进行熟悉，以下终结一些自己常用的语句规则，方便日后查看，因为会不断用到新的格式，所以也会不定时地更新






1.标题

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

大标题可以直接在下面加=
=

小标题加-
-
